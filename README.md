# 🏥 Healthcare Provider Dashboard | Power BI

## 📌 Overview
This project is an interactive Power BI dashboard designed to analyze healthcare costs, billing performance, and service distribution.

## 📊 Key Metrics
- Total Billing Amount
- Medication Cost
- Treatment Cost
- Insurance Coverage
- Out-of-Pocket Expenses
- Room Charges

## 📈 Features
- Billing analysis by procedure and department
- Diagnosis & service type breakdown
- Geographic visualization of patient distribution
- Light & Dark mode dashboard design

## 🛠 Tools Used
- Power BI
- DAX
- Data Modeling
- Data Visualization
